# AI Invoice Analysis Agent using LangGraph and Gemini

## Overview

This project is a simple AI-powered invoice analysis agent built using LangGraph and Gemini (via OpenRouter).

The agent performs:

* Invoice Validation
* Risk Analysis
* AI-Based Invoice Assessment
* Automated Report Generation

## Workflow

Invoice Input
→ Validation Node
→ Risk Analysis Node
→ Gemini Analysis Node
→ Report Generation Node

## Technologies Used

* Python
* LangGraph
* Gemini 2.5 Flash
* OpenRouter
* Google Colab

## Features

* State-based workflow
* Risk scoring mechanism
* LLM-powered invoice insights
* End-to-end automation
